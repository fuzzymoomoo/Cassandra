# Cassandra

> **Don't just be right. Bring the evidence.**

Cassandra is a learner-led research companion: browse, deliberately capture small evidence, explain why it matters, compare sources, then ask AI to help synthesize the packet.

## Status

Wave 3 is complete: Cassandra now supports deliberate captures, a source-grouped evidence trail, research-packet export/backup, and an input-only GPT-5.6 courier. It never submits a prompt or reads an AI response.

## Development

See [BUILD.md](BUILD.md) for commands and [HANDOFF.md](HANDOFF.md) for the next wave.

## Privacy boundary

The MVP will only capture after an explicit learner action. It excludes form controls, editable content, and private-account surfaces; it never reads ChatGPT responses or submits prompts automatically.
